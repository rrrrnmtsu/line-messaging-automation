## ステップ2: 最初の課題（10分）

### プログラミング未経験の方
1. Claude Codeを起動
2. 以下のコマンドを実行:
```bash
# ディレクトリ作成
mkdir my-first-project
cd my-first-project

# Claude Codeに依頼
"Hello World を表示するPythonプログラムを作成してください"
```

### プログラミング経験者の方
1. Claude Codeを起動
2. 以下を実装:
```bash
"CSVファイルを読み込んで、データを集計するPythonスクリプトを作成してください。
- 合計、平均、最大値、最小値を計算
- 結果をコンソールに表示"
```

## ステップ3: 今週の目標設定（5分）

以下のチェックリストから3つ選んで、今週中に達成しましょう：

- [ ] Claude Codeで5つのスクリプトを作成
- [ ] エラーを3回解決
- [ ] [TASKS.md](./TASKS.md) のWeek 1タスクを50%完了
- [ ] 自分の業務で使える自動化ツールを1つ作成
- [ ] [ROADMAP.md](./ROADMAP.md) を読んで学習計画を立てる

---

## ステップ4: 継続のコツ

### 毎日のルーティン（15分）
1. Claude Codeを起動
2. 前日のコードをレビュー
3. 新しいタスクを1つ実行
4. 学習メモを記録

### 週1回のレビュー（30分）
1. 達成したタスクを確認
2. 学んだことを3つ書き出す
3. 来週の目標を設定

---
