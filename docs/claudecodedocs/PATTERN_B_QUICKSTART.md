- 感覚ではなく数値で判断

## 🚨 よくあるトラブルと対処法

### トラブル1: WordPress REST APIが動かない
**原因**: アプリケーションパスワード未設定
**対処**: WordPressダッシュボード → ユーザー → プロフィール → アプリケーションパスワード

### トラブル2: 記事の品質が低い
**原因**: プロンプトが曖昧
**対処**: 具体例を3つ示す、文字数・構成を明確化

### トラブル3: PVが全然増えない
**原因**: インデックス未完了、競合が強すぎる
**対処**: Search Consoleでインデックス確認、ロングテールキーワード狙い

### トラブル4: 時間がない
**原因**: 完璧主義、優先順位が不明確
**対処**: 週の最初に優先タスク3つを決める、時間制限を設ける
