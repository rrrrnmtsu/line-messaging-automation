## 書き出しガイドライン

### 推奨フォーマット

| 用途 | フォーマット | サイズ | 備考 |
|-----|------------|--------|------|
| ロゴ（Web） | PNG（透過） | 300x300px | 背景透過必須 |
| ロゴ（ベクター） | SVG | - | スケーラブル |
| UI背景 | JPG | 1920x1080px | 品質90% |
| アイコン | SVG/PNG | 24x24, 48x48px | 複数サイズ |
| OGP画像 | JPG/PNG | 1200x630px | SNS共有用 |

### 最適化

- PNG: [TinyPNG](https://tinypng.com/) で圧縮
- JPG: 品質80-90%推奨
- SVG: 不要なメタデータ削除
