# Task Completion Checklist
- Implement requested changes and ensure TypeScript builds (`npm run build`) when relevant.
- Run targeted Jest tests (`npm test`) if code paths have coverage.
- Run `npm run lint` when modifying TypeScript logic to maintain linter compliance.
- Update documentation or schema files if interfaces/outputs change.
- Summarize changes and follow collaboration protocols (update `PROGRESS.md`, drop completion flags) before handoff.
