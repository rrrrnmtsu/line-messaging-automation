# Suggested Commands
- `npm install` — install dependencies after cloning.
- `npm run dev -- --phase 1` — run CLI in development mode for phase 1 harvesting.
- `npm run dev -- --phase 2 --focus ...` — run phase 2 with focus industries.
- `npm run build` — compile TypeScript to `dist/`.
- `npm start` — execute compiled CLI (`dist/cli.js`).
- `npm test` — execute Jest test suite.
- `npm run lint` — lint TypeScript sources with ESLint.
